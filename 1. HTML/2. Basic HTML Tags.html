0. DOCTYPE: Used to tell the browser that this is an HTML 5 document and render it as a HTML 5 document only
1. html: Also called as root tag includes all the other tags into it. There can be only one body tag in whole html document
2. head: Used as a container for metadata (data about data) and is placed between the html tag and the body
3. meta: Used to store metadata about an HTML document. Metadata is data about the data.
Metadata typically define the document title, character set, styles, scripts, and other meta information.
3.1 meta-charset: specifies the character encoding for the HTML document UTF-8 is a default character set for html
and contains all the punctuations and characters for use, included in UTF-8 charset.
3.2 meta-viewport: The viewport is the user's visible area of a web page. It varies with the device - it will be smaller
on a mobile phone than on a computer screen. The width=device-width part sets the width of the page to follow
the screen-width of the device (which will vary depending on the device). The initial-scale=1.0 part sets the initial zoom level when the page is first loaded by the browser.
3.3 meta-description:Used to define a description of your web page for better SEO,
3.3 meta-keywords: Used to define keywords for search engines foor better SEO.
3.3 meta-author: Define the author of a page for better SEO.
3.4 meta-robots: For some reasons you dont't wanna rank your site on search engines then use robots meta tag and NOINDEX for not indexing in search engines and NOFOLLOW for not follwoing any links on your web page.
4. title: Used to defines the title of the document. The title must be text-only, and it is shown in the browser's title bar
or in the page's tab.
5. body: Used to defines the document's body. The body tag element contains all the contents of an HTML document, such as headings,
paragraphs, images, hyperlinks, tables, lists, etc. There can be only one body tag in whole html document.
6. div: The div tag defines a division or a section in an HTML document. The div tag is used as a container for HTML elements - which is
then styled with CSS or manipulated with JavaScript.
