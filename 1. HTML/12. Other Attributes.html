Before understanding any attribute lets understand first what's attribute is and it's types:

Attribute:

An attribute is a keyword used to manipulate an html element to customize it more, in terms of looks and functionality.

Types of attributes:

There are two types of attributes in html they are:

  • Global Attribute: Works with only some specific HTML element.
  • Element Attribute: Works with any HTML element.

Now let's cover each attribute

id: Specifies a unique id for an element

class: Specifies one or more classnames for an element.

style: Specifies an inline CSS style for an element

rel: Specifies the relationship between the current document and the linked document

download: to give a functionality to dowload something to an HTML element.

readonly: Specifies that the element is read-only

disabled: Specifies that the specified element/group of elements should be disabled and stop working it's functioning.

autofocus: Specifies that the element should automatically get focus when the page loads.

autocomplete: Specifies whether the form or the input element should have autocomplete enabled

multiple: Specifies that a user can enter more than one value in input or selecct tags only.

contenteditable: makes any content editable if value is "true"
