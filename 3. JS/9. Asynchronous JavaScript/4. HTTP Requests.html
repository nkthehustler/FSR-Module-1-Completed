HTTP Requests

A HTTP Request stads for hyper text transfer protocol, bcasially its a tytpe f protocol to transfer the data between server and client

HTTP Requests used for:

  1. Sending and receiving data
  2. You send a Requests as a Client and Server Responds with data
  3. The data can be anything such as , HTML, CSS, JS files or images, audio amd media files, ass well as formated data such as JSON or XML

HTTP Methods used to create HTTP requests they are:

  1. GET: Fetch / Retrive data from the server to the client
  2. POST: Send data from client to the server
  3. PUT & PATCH: Update the data on the server
  4. DELETE: Delete the data on the server

HTTP Codes to detect the type of response:

  1. 100 Range: Continue
  2. 200 Range: Success
  3. 300 Range: Redirects
  4. 400 Range: Client Side Error
  5. 500 Range: Server Side Error
